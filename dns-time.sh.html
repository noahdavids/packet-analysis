dns-time.sh Information

This macro uses tshark to find DNS (port 53) queries and calculate the time between query and response. It differs from the dns.time value caculated by wireshark in that it calculates the time between the first response for transaction ID X and the first query for transaction X not the first response and last query for transaction X.

The output is a a table with the following columns.
    Server ID Type Name Response-time - Query-time = Delta-time
Where:

  • server
         IP address of the server the request is sent to
  • ID
         This is the DNS ID of the query
  • Type
         The type of query. The common ones are
    • 1       IPv4 Name query
    • 28     IPv6 Name query
    • 12     Reverse IPv4 Name query (map an IP address back to a name)
  • Name
         Name being queries
  • Response-time
         Packet arrivial time of the first response
  • Query-time
         Packet arrival time of the first query (when the query was sent)
  • Delta-time
         The time between Respons-time and Query-time

It also lists unanswered queries in the table
     Server ID Type Name Query-time

Usage

dns-time.sh FILE-NAME

FILE-NAME
The file name (or path to the file), This file must be reabale by tshark.

Examples

 Example 1 - Execute dns-time.sh 
                                                                   
$ ./dns-time.sh /cloud/seagate/packet-trace-library/dns-2.pcap
